# andor-pwa
Andor-Legenden App (PWA with Vue.js)

https://andor-pwa.firebaseapp.com/


### Build local
Create all files for the app in hosting/dist
```console 
cd hosting 
npm i 
npm run build
```

### Dev local
Start local Server with Hot-Module-Reloading
```console 
cd hosting 
npm i 
npm run dev
```
